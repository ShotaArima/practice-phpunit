## PHPUnitとは
- PHP言語の単体テストを行うためのフレームワーク
- 変更した箇所で要件を満たしていて、既存の機能に影響がないかを毎回確認することを自動化できる

###流れ
1. PHPUnitをComposerで入れる
2. テストコードを書く
3. コード実行で利用
